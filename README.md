# 🥂 LUXIP - 프리미엄 주류 스마트 오더 사이트

> **LUXIP**은 고급스러운 주류를 스마트하게 주문하고,  
> 매장 위치부터 결제까지 한 번에 처리할 수 있는 술 판매 플랫폼입니다. 🍷✨

---

## 🛠️ 사용 기술 스택

- **Frontend**: React, HTML5, CSS3, JavaScript, vscode, axios  등
- **Backend**: 없음 (프론트 중심 SPA)
- **DB**: JSON 파일 기반 데이터 관리
- **지도 서비스**: Kakao Map API
- **버전관리**: Git / GitHub

---

## ⚙️ 주요 기능

### 🧑‍💼 관리자 기능 (허린)
- 🖼️ **관리자 전용 레이아웃**
- 👥 **회원 관리**
- 🍶 **상품 관리**
- 🚚 **주문처 관리**
- 📊 **인덱스 대시보드 / 메인 페이지**
- 🗺️ **카카오 API 기반 매장 지도 연동**
- 🧾 **GitHub 버전 관리 총괄**

### 💳 주문 / 결제 기능 (박한나)
- 🧺 **장바구니 기능**
- 📦 **주문 관리 페이지**
- 💳 **결제 처리 시스템**

### 🛍️ 사용자 기능 (이승준)
- 📦 **아이템 목록 페이지**: 전체 주류 상품 리스트 확인
- 🔍 **아이템 상세 페이지**: 이미지, 설명, 댓글 기능 포함
- 📝 **로그인 / 회원가입**: 사용자 인증 기능
